# 🦠 孟加拉国 COVID-19 疫情数据分析（R 语言）

## 📌 项目简介

本项目使用 R 语言对孟加拉国 2020 年 3 月至 5 月的新冠疫情数据进行探索性分析，涵盖数据清洗、可视化与统计推断，旨在揭示疫情传播趋势、峰值特征及防控效果。

**项目目标**：通过多维度分析（时间趋势、相关性、阶段对比），为疫情防控决策提供数据支持。

## 📊 数据概览

- **数据来源**：COVID-19 BD Dataset（孟加拉国官方疫情数据）
- **时间范围**：2020-03-08 至 2020-05-05
- **核心指标**：每日/累计确诊、死亡、治愈、检测数

## 🔍 分析问题

本项目围绕以下 5 个核心问题展开分析：

1. 每日新增确诊/死亡/治愈的趋势变化及峰值出现时间？
2. 累计确诊/死亡/治愈的增长曲线特征（指数型 vs 线性）？
3. 每日新增确诊数与每日新增死亡数是否存在正相关关系？
4. 疫情爆发初期与后期的每日新增确诊数差异是否显著？
5. 累计死亡数占累计确诊数的比例随时间如何变化？

## 📈 核心发现

1. **新增趋势**：每日新增确诊在 4 月中旬达到峰值，随后趋于平稳
2. **累计增长**：累计确诊呈指数增长，死亡和治愈增长相对平缓
3. **相关性**：新增确诊与新增死亡呈正相关关系
4. **阶段差异**：疫情后期新增确诊显著高于爆发初期
5. **致死率**：累计死亡占比整体呈下降趋势，维持在较低水平
